# math abs

**version**: 0.85.0

## **usage**:

Returns the absolute value of a number.

## Signature

`> math abs `

## Input/output types:

| input            | output           |
| ---------------- | ---------------- |
| duration         | duration         |
| list\<duration\> | list\<duration\> |
| list\<number\>   | list\<number\>   |
| number           | number           |

## Examples

Compute absolute value of each number in a list of numbers

```bash
> [-50 -100.0 25] | math abs
```
