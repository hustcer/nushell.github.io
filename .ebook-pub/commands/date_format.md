# date format

**version**: 0.100.1

## **usage**:

Removed command: use `format date` instead.

## Signature

`> date format (format string) --list`

## Parameters

- `format string`: The desired date format.
- `--list`: lists strftime cheatsheet

## Input/output types:

| input    | output |
| -------- | ------ |
| datetime | string |
| string   | string |
