# math product

**version**: 0.85.0

## **usage**:

Returns the product of a list of numbers or the products of each column of a table.

## Signature

`> math product `

## Input/output types:

| input          | output |
| -------------- | ------ |
| list\<number\> | number |

## Examples

Compute the product of a list of numbers

```bash
> [2 3 3 4] | math product
```
