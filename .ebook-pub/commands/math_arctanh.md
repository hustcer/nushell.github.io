# math arctanh

**version**: 0.93.0

## **usage**:

Returns the inverse of the hyperbolic tangent function.

## Signature

`> math arctanh `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |

## Examples

Get the arctanh of 1

```bash
> 1 | math arctanh
```
