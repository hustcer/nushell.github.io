# math arctanh

**version**: 0.85.0

## **usage**:

Returns the inverse of the hyperbolic tangent function.

## Signature

`> math arctanh `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Get the arctanh of 1

```bash
> 1 | math arctanh
```
