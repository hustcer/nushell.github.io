# math log

**version**: 0.80.0

## **usage**:

Returns the logarithm for an arbitrary base.

## Signature

`> math log (base)`

## Parameters

- `base`: Base for which the logarithm should be computed

## Examples

Get the logarithm of 100 to the base 10

```bash
> 100 | math log 10
```

Get the log2 of a list of values

```bash
> [16 8 4] | math log 2
```
