# roll up

**version**: 0.93.0

## **usage**:

Roll table rows up.

## Signature

`> roll up --by`

## Parameters

- `--by {int}`: Number of rows to roll

## Input/output types:

| input | output |
| ----- | ------ |
| table | table  |

## Examples

Rolls rows up

```bash
> [[a b]; [1 2] [3 4] [5 6]] | roll up
```
