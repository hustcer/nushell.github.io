# to yaml

**version**: 0.85.0

## **usage**:

Convert table into .yaml/.yml text.

## Signature

`> to yaml `

## Input/output types:

| input | output |
| ----- | ------ |
| any   | string |

## Examples

Outputs an YAML string representing the contents of this table

```bash
> [[foo bar]; ["1" "2"]] | to yaml
```
