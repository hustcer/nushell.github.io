# clear

**version**: 0.93.0

## **usage**:

Clear the terminal.

## Signature

`> clear --all`

## Parameters

- `--all`: Clear the terminal and its scroll-back history

## Input/output types:

| input   | output  |
| ------- | ------- |
| nothing | nothing |

## Examples

Clear the terminal

```bash
> clear
```

Clear the terminal and its scroll-back history

```bash
> clear --all
```
