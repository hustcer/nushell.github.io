# panic

**version**: 0.100.1

## **usage**:

Causes nushell to panic.

## Signature

`> panic (msg)`

## Parameters

- `msg`: The custom message for the panic.

## Input/output types:

| input   | output  |
| ------- | ------- |
| nothing | nothing |

## Examples

Panic with a custom message

```bash
> panic 'This is a custom panic message'
```
