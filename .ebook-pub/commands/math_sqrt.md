# math sqrt

**version**: 0.93.0

## **usage**:

Returns the square root of the input number.

## Signature

`> math sqrt `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |

## Examples

Compute the square root of each number in a list

```bash
> [9 16] | math sqrt
```
