# math sqrt

**version**: 0.85.0

## **usage**:

Returns the square root of the input number.

## Signature

`> math sqrt `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Compute the square root of each number in a list

```bash
> [9 16] | math sqrt
```
