# roll down

**version**: 0.90.2

## **usage**:

Roll table rows down.

## Signature

`> roll down --by`

## Parameters

- `--by {int}`: Number of rows to roll

## Input/output types:

| input | output |
| ----- | ------ |
| table | table  |

## Examples

Rolls rows down of a table

```bash
> [[a b]; [1 2] [3 4] [5 6]] | roll down
```
