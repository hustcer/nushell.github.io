# dfr col

**version**: 0.90.2

## **usage**:

Creates a named column expression.

## Signature

`> dfr col (column name)`

## Parameters

- `column name`: Name of column to be used

## Input/output types:

| input | output |
| ----- | ------ |
| any   | any    |

## Examples

Creates a named column expression and converts it to a nu object

```bash
> dfr col a | dfr into-nu
```
