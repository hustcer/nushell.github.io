# let-env

**version**: 0.93.0

## **usage**:

`let-env FOO = ...` has been removed, use `$env.FOO = ...` instead.

## Signature

`> let-env (var_name) (initial_value)`

## Parameters

- `var_name`: Variable name.
- `initial_value`: Equals sign followed by value.

## Input/output types:

| input   | output  |
| ------- | ------- |
| nothing | nothing |
