# dfr uppercase

**version**: 0.93.0

## **usage**:

Uppercase the strings in the column.

## Signature

`> dfr uppercase `

## Input/output types:

| input | output |
| ----- | ------ |
| any   | any    |

## Examples

Modifies strings to uppercase

```bash
> [Abc aBc abC] | dfr into-df | dfr uppercase
```
