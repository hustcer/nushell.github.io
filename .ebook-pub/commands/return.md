# return

**version**: 0.80.0

## **usage**:

Return early from a function.

## Signature

`> return (return_value)`

## Parameters

- `return_value`: optional value to return

## Notes

```text
This command is a parser keyword. For details, check:
  https://www.nushell.sh/book/thinking_in_nu.html
```

## Examples

Return early

```bash
> def foo [] { return }
```
