# return

**version**: 0.93.0

## **usage**:

Return early from a function.

## Signature

`> return (return_value)`

## Parameters

- `return_value`: Optional value to return.

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | any    |

## Examples

Return early

```bash
> def foo [] { return }
```
