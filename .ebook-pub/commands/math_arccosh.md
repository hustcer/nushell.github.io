# math arccosh

**version**: 0.85.0

## **usage**:

Returns the inverse of the hyperbolic cosine function.

## Signature

`> math arccosh `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Get the arccosh of 1

```bash
> 1 | math arccosh
```
