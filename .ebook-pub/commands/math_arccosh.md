# math arccosh

**version**: 0.90.2

## **usage**:

Returns the inverse of the hyperbolic cosine function.

## Signature

`> math arccosh `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |

## Examples

Get the arccosh of 1

```bash
> 1 | math arccosh
```
