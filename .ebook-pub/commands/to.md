# to

**version**: 0.100.1

## **usage**:

Translate structured data to a format.

## Signature

`> to `

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | string |

## Notes

```text
You must use one of the following subcommands. Using this command as-is will only produce this help message.
```
