# debug info

**version**: 0.90.2

## **usage**:

View process memory info.

## Signature

`> debug info `

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | record |

## Examples

View process information

```bash
> debug info
```

## Notes

```text
This command is meant for debugging purposes.
It shows you the process information and system memory information.
```
