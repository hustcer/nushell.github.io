# tutor

**version**: 0.80.0

## **usage**:

Run the tutorial. To begin, run: tutor.

## Signature

`> tutor (search) --find`

## Parameters

- `search`: item to search for, or 'list' to list available tutorials
- `--find {string}`: Search tutorial for a phrase

## Examples

Begin the tutorial

```bash
> tutor begin
```

Search a tutorial by phrase

```bash
> tutor -f "$in"
```
