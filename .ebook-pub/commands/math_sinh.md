# math sinh

**version**: 0.85.0

## **usage**:

Returns the hyperbolic sine of the number.

## Signature

`> math sinh `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Apply the hyperbolic sine to 1

```bash
> 1 | math sinh
```
