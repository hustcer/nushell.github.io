# math sinh

**version**: 0.100.1

## **usage**:

Returns the hyperbolic sine of the number.

## Signature

`> math sinh `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |

## Examples

Apply the hyperbolic sine to 1

```bash
> 1 | math sinh
```
