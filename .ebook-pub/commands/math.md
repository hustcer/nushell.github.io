# math

**version**: 0.100.1

## **usage**:

Use mathematical functions as aggregate functions on a list of numbers or tables.

## Signature

`> math `

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | string |

## Notes

```text
You must use one of the following subcommands. Using this command as-is will only produce this help message.
```
