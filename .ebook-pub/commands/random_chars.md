# random chars

**version**: 0.85.0

## **usage**:

Generate random chars.

## Signature

`> random chars --length`

## Parameters

- `--length {int}`: Number of chars

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | string |

## Examples

Generate random chars

```bash
> random chars
```

Generate random chars with specified length

```bash
> random chars -l 20
```
