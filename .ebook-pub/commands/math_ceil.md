# math ceil

**version**: 0.85.0

## **usage**:

Returns the ceil of a number (smallest integer greater than or equal to that number).

## Signature

`> math ceil `

## Input/output types:

| input          | output      |
| -------------- | ----------- |
| list\<number\> | list\<int\> |
| number         | int         |

## Examples

Apply the ceil function to a list of numbers

```bash
> [1.5 2.3 -3.1] | math ceil
```
