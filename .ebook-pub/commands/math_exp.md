# math exp

**version**: 0.100.1

## **usage**:

Returns e raised to the power of x.

## Signature

`> math exp `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |

## Examples

Get e raised to the power of zero

```bash
> 0 | math exp
```

Get e (same as 'math e')

```bash
> 1 | math exp
```
