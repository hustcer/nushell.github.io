# random binary

**version**: 0.100.1

## **usage**:

Generate random bytes.

## Signature

`> random binary (length)`

## Parameters

- `length`: Length of the output binary.

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | binary |

## Examples

Generate 16 random bytes

```bash
> random binary 16
```

Generate 1 random kilobyte

```bash
> random binary 1kb
```
