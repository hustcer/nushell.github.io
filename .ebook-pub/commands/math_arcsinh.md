# math arcsinh

**version**: 0.85.0

## **usage**:

Returns the inverse of the hyperbolic sine function.

## Signature

`> math arcsinh `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Get the arcsinh of 0

```bash
> 0 | math arcsinh
```
