# view files

**version**: 0.85.0

## **usage**:

View the files registered in nushell's EngineState memory.

## Signature

`> view files `

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | string |

## Examples

View the files registered in nushell's EngineState memory

```bash
> view files
```

## Notes

```text
These are files parsed and loaded at runtime.
```
