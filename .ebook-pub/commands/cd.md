# cd

**version**: 0.90.2

## **usage**:

Change directory.

## Signature

`> cd (path)`

## Parameters

- `path`: The path to change to.

## Input/output types:

| input   | output  |
| ------- | ------- |
| nothing | nothing |
| string  | nothing |

## Examples

Change to your home directory

```bash
> cd ~
```

Change to the previous working directory ($OLDPWD)

```bash
> cd -
```
