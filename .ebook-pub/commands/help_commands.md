# help commands

**version**: 0.93.0

## **usage**:

Show help on nushell commands.

## Signature

`> help commands ...rest --find`

## Parameters

- `...rest`: The name of command to get help on.
- `--find {string}`: string to find in command names, usage, and search terms

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | table  |
