# dfr lit

**version**: 0.90.2

## **usage**:

Creates a literal expression.

## Signature

`> dfr lit (literal)`

## Parameters

- `literal`: literal to construct the expression

## Input/output types:

| input | output |
| ----- | ------ |
| any   | any    |

## Examples

Created a literal expression and converts it to a nu object

```bash
> dfr lit 2 | dfr into-nu
```
