# dfr shape

**version**: 0.90.2

## **usage**:

Shows column and row size for a dataframe.

## Signature

`> dfr shape `

## Input/output types:

| input | output |
| ----- | ------ |
| any   | any    |

## Examples

Shows row and column shape

```bash
> [[a b]; [1 2] [3 4]] | dfr into-df | dfr shape
```
