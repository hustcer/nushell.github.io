# into

**version**: 0.100.1

## **usage**:

Commands to convert data from one type to another.

## Signature

`> into `

## Input/output types:

| input   | output |
| ------- | ------ |
| nothing | string |

## Notes

```text
You must use one of the following subcommands. Using this command as-is will only produce this help message.
```
