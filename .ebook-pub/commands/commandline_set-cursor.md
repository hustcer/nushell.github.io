# commandline set-cursor

**version**: 0.100.1

## **usage**:

Set the current cursor position.

## Signature

`> commandline set-cursor (pos) --end`

## Parameters

- `pos`: Cursor position to be set
- `--end`: set the current cursor position to the end of the buffer

## Input/output types:

| input   | output  |
| ------- | ------- |
| nothing | nothing |
