# math ln

**version**: 0.100.1

## **usage**:

Returns the natural logarithm. Base: (math e).

## Signature

`> math ln `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |

## Examples

Get the natural logarithm of e

```bash
> 2.7182818 | math ln | math round --precision 4
```
