# math ln

**version**: 0.85.0

## **usage**:

Returns the natural logarithm. Base: (math e).

## Signature

`> math ln `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Get the natural logarithm of e

```bash
> 2.7182818 | math ln | math round --precision 4
```
