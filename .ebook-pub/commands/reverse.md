# reverse

**version**: 0.90.2

## **usage**:

Reverses the input list or table.

## Signature

`> reverse `

## Input/output types:

| input       | output      |
| ----------- | ----------- |
| list\<any\> | list\<any\> |

## Examples

Reverse a list

```bash
> [0,1,2,3] | reverse
```

Reverse a table

```bash
> [{a: 1} {a: 2}] | reverse
```
