# to plist

**version**: 0.100.1

## **usage**:

Convert Nu values into plist

## Signature

`> to plist --binary`

## Parameters

- `--binary`: Output plist in binary format

## Input/output types:

| input | output |
| ----- | ------ |
| any   | any    |

## Examples

Convert a table into a plist file

```bash
> { a: 3 } | to plist
```
