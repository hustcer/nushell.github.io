# math tanh

**version**: 0.90.2

## **usage**:

Returns the hyperbolic tangent of the number.

## Signature

`> math tanh `

## Input/output types:

| input          | output        |
| -------------- | ------------- |
| list\<number\> | list\<float\> |
| number         | float         |
