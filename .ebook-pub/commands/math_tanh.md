# math tanh

**version**: 0.85.0

## **usage**:

Returns the hyperbolic tangent of the number.

## Signature

`> math tanh `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |
