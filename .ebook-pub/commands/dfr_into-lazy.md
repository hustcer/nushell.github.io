# dfr into-lazy

**version**: 0.85.0

## **usage**:

Converts a dataframe into a lazy dataframe.

## Signature

`> dfr into-lazy `

## Input/output types:

| input | output |
| ----- | ------ |
| any   | any    |

## Examples

Takes a dictionary and creates a lazy dataframe

```bash
> [[a b];[1 2] [3 4]] | dfr into-lazy
```
