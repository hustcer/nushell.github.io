# math cosh

**version**: 0.85.0

## **usage**:

Returns the hyperbolic cosine of the number.

## Signature

`> math cosh `

## Input/output types:

| input          | output         |
| -------------- | -------------- |
| list\<number\> | list\<number\> |
| number         | number         |

## Examples

Apply the hyperbolic cosine to 1

```bash
> 1 | math cosh
```
