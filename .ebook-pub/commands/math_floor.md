# math floor

**version**: 0.93.0

## **usage**:

Returns the floor of a number (largest integer less than or equal to that number).

## Signature

`> math floor `

## Input/output types:

| input          | output      |
| -------------- | ----------- |
| list\<number\> | list\<int\> |
| number         | int         |

## Examples

Apply the floor function to a list of numbers

```bash
> [1.5 2.3 -3.1] | math floor
```
