# bits and

**version**: 0.80.0

## **usage**:

Performs bitwise and for integers.

## Signature

`> bits and (target)`

## Parameters

- `target`: target integer to perform bit and

## Examples

Apply bits and to two numbers

```bash
> 2 | bits and 2
```

Apply logical and to a list of numbers

```bash
> [4 3 2] | bits and 2
```
