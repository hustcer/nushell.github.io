# from msgpackz

**version**: 0.100.1

## **usage**:

Convert brotli-compressed MessagePack data into Nu values.

## Signature

`> from msgpackz --objects`

## Parameters

- `--objects`: Read multiple objects from input

## Input/output types:

| input  | output |
| ------ | ------ |
| binary | any    |

## Notes

```text
This is the format used by the plugin registry file ($nu.plugin-path).
```
