# math arctan

**version**: 0.80.0

## **usage**:

Returns the arctangent of the number.

## Signature

`> math arctan --degrees`

## Parameters

- `--degrees`: Return degrees instead of radians

## Examples

Get the arctangent of 1

```bash
> 1 | math arctan
```

Get the arctangent of -1 in degrees

```bash
> -1 | math arctan -d
```
