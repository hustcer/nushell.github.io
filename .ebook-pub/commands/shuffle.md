# shuffle

**version**: 0.85.0

## **usage**:

Shuffle rows randomly.

## Signature

`> shuffle `

## Input/output types:

| input       | output      |
| ----------- | ----------- |
| list\<any\> | list\<any\> |

## Examples

Shuffle rows randomly (execute it several times and see the difference)

```bash
> [[version patch]; ['1.0.0' false] ['3.0.1' true] ['2.0.0' false]] | shuffle
```
