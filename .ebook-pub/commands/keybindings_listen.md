# keybindings listen

**version**: 0.93.0

## **usage**:

Get input from the user.

## Signature

`> keybindings listen `

## Input/output types:

| input   | output  |
| ------- | ------- |
| nothing | nothing |

## Examples

Type and see key event codes

```bash
> keybindings listen
```

## Notes

```text
This is an internal debugging tool. For better output, try `input listen --types [key]`
```
